
# Debate Guardian

Real-time fact-checking for honest discussions

## Project Overview

Debate Guardian is an advanced debate fact-checking system that analyzes speech in real-time to identify claims, verify facts, and assess logical fallacies. The application helps maintain honesty and accountability in discussions, debates, and public speaking.

## Key Features

- **Real-time Speech Analysis**: Captures and analyzes speech as it happens
- **Claim Detection**: Automatically identifies claims that need fact-checking
- **Fact Verification**: Checks claims against a database of facts and provides verdicts
- **Logical Fallacy Detection**: Identifies logical fallacies in arguments
- **Credibility Scoring**: Provides balanced scoring for claims based on verification status
- **Emotion Detection**: Optional analysis of speaker emotions during debates
- **Adjustable Tolerance**: Configurable strictness for fact-matching
- **Speaker Statistics**: Tracks speaker performance, claim accuracy, and argumentation style
- **Analytics Visualization**: Graphs and charts showing debate trends and patterns

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the development server with `npm run dev`
4. Set up your Gemini AI API key for enhanced fact-checking

## Using the Application

1. Select a speaker by clicking their card
2. Click "Start Listening" to begin capturing speech
3. Make claims to see them fact-checked in real-time
4. Use the tolerance slider to adjust fact-checking strictness
5. Enable emotion detection for additional insights
6. View speaker statistics and analytics in their respective tabs
